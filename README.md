# Annoyng VIM plugin

# WARNING THIS IS ANNOYING

## How to use
1. Add
```vim
call plug#begin()
  Plug 'mousegames/AnnoyingVimPlugin'
call plug#end()

```
to your .vimrc if you havn't already

2. in vim run
```vim
:Pluginstall
```

3. to use run
```vim
:Annoying
```
to stop run
```vim
:AnnoyingOff
```

## depandencies
- vim-plug
- a working .vimrc
